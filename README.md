# Vibe Data Automation Landing Page

A modern, single-page SaaS landing experience for Vibe, an AI-powered data automation platform. Built with React, Vite, Tailwind CSS, Shadcn-inspired UI components, Framer Motion animations, and Lucide icons.

## Getting started

```bash
npm install
npm run dev
```

If your environment blocks access to npm registries, install dependencies from an approved mirror before running the app.

## Available scripts

- `npm run dev` – start the Vite dev server.
- `npm run build` – type-check and build for production.
- `npm run preview` – preview the production build locally.

## Features

- Hero with gradient/glassmorphism styling and enterprise CTAs.
- Animated feature cards for the core services.
- Social proof, workflow, testimonials, and pricing preview sections.
- Responsive layout with smooth Framer Motion scroll reveals.
